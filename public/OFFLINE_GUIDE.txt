
راهنمای اجرای نرم‌افزار در محیط آفلاین (کارخانه)
================================================

با توجه به محدودیت دسترسی به اینترنت در کارخانه، برای اجرای این نرم‌افزار از یکی از دو روش زیر استفاده کنید.

روش پیشنهادی: انتقال نسخه نهایی (Build)
---------------------------------------
این روش سریع‌ترین و سبک‌ترین راه است و نیازی به نصب Node.js یا پکیج‌ها روی سرور کارخانه ندارد.

1. در یک سیستم دارای اینترنت:
   - پروژه را باز کنید.
   - دستور npm install را اجرا کنید تا پکیج‌ها دانلود شوند.
   - دستور npm run build را اجرا کنید.
   - پس از اتمام، یک پوشه جدید به نام dist ساخته می‌شود.

2. در سرور کارخانه (آفلاین):
   - کل پوشه dist را به سرور منتقل کنید.
   - این پوشه حاوی فایل‌های index.html و فایل‌های جاوااسکریپت یکپارچه شده است.
   - این فایل‌ها را می‌توانید با هر وب‌سروری (مثل IIS، Apache، Nginx یا حتی افزونه Serve) اجرا کنید.

---------------------------------------

روش توسعه: انتقال دستی پکیج‌ها
------------------------------
اگر نیاز دارید کدها را در محیط آفلاین ویرایش کنید:

1. در سیستم دارای اینترنت:
   - دستور npm install را اجرا کنید.
   - پوشه node_modules ایجاد شده را کپی کنید.

2. در سیستم آفلاین:
   - پوشه node_modules را در ریشه پروژه قرار دهید.
   - نکته مهم: فایل index.html فعلی برای استفاده سریع از اینترنت (CDN) تنظیم شده است. برای حالت آفلاین، باید تگ <script type="importmap">...</script> را از فایل index.html حذف کنید تا برنامه به جای اینترنت، از پوشه node_modules استفاده کند.

فونت‌ها
-------
مطمئن شوید که فایل‌های فونت (طبق راهنمای پوشه public/fonts) در جای خود قرار گرفته‌اند، در غیر این صورت فونت‌ها در حالت آفلاین به هم می‌ریزند.
